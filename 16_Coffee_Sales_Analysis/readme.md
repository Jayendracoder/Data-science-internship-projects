# Coffee Sales Analysis

## Project Overview
This project analyzes coffee sales data to understand customer preferences,
sales trends, and revenue drivers across locations and time periods.

## Problem Statement
Coffee businesses need insights to optimize pricing, inventory, and promotions.

## Dataset Description
- Date
- Coffee type
- Quantity sold
- Revenue
- Store location

## Analysis Performed
- Daily & monthly sales trends
- Product-wise performance
- Location-wise revenue analysis

## Machine Learning
- Sales prediction using regression

## Tableau Dashboard
- Sales trends
- Product performance
- Store comparison
