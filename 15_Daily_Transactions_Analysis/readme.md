# Daily Transactions Analysis

## Project Overview
This project analyzes daily transaction data to understand spending behavior,
transaction volume, and revenue trends.

## Problem Statement
Businesses need transaction insights to detect anomalies and optimize operations.

## Dataset Description
- Transaction date
- Amount
- Category
- Payment method

## Analysis Performed
- Daily & monthly transaction trends
- Category-wise spending
- Anomaly detection

## Machine Learning
- Transaction amount prediction

## Tableau Dashboard
- Daily transaction trends
- Category-wise spending
