# World Population Analysis

## Project Overview
This project analyzes global population data to understand population growth,
country-wise distribution, and demographic trends over time.

## Problem Statement
Governments and organizations require population insights for planning resources,
infrastructure, and policy-making.

## Dataset Description
- Country
- Population
- Year
- Growth rate
- Density

## Analysis Performed
- Population growth trends
- Country-wise comparison
- Regional population distribution

## Machine Learning
- Regression for population growth forecasting

## Tableau Dashboard
- Population by country
- Growth trends over years
- Regional distribution
