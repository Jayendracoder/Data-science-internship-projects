# Twitter Financial News Analysis

## Project Overview
This project analyzes financial news tweets to understand sentiment trends
and their impact on market behavior.

## Problem Statement
Market sentiment on social media influences investor decisions and volatility.

## Dataset Description
- Tweet text
- Date
- Sentiment score
- Related stock/sector

## Analysis Performed
- Sentiment analysis
- Trend detection
- Market correlation analysis

## Machine Learning
- Classification for sentiment prediction

## Tableau Dashboard
- Sentiment distribution
- Trend over time
