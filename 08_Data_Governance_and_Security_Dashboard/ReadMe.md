# Data Governance and Security Dashboard

## Project Overview
This project focuses on analyzing organizational data governance, data quality,
access control, and security risks. The goal is to provide insights into data
compliance, integrity, and security monitoring using analytics and dashboards.

## Problem Statement
Organizations must ensure data privacy, compliance, and secure access to data.
Poor governance leads to data breaches, compliance issues, and operational risks.

## Dataset Description
- Data asset type
- Access level
- Data owner
- Compliance status
- Risk category
- Audit logs

## Tools & Technologies
- Python (Pandas, NumPy)
- Data Analysis & EDA
- Tableau Dashboards

## Analysis Performed
- Data access pattern analysis
- Risk and compliance monitoring
- Data quality checks

## Tableau Dashboard
- Data risk classification
- Access-level distribution
- Compliance status overview
