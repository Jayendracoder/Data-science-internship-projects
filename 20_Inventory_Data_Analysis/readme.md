# Inventory Data Analysis

## Project Overview
This project analyzes inventory data to optimize stock levels, reduce holding
costs, and prevent stockouts.

## Problem Statement
Poor inventory management leads to lost sales and increased costs.

## Dataset Description
- Product ID
- Stock level
- Reorder point
- Demand
- Lead time

## Analysis Performed
- Stock utilization analysis
- Demand vs inventory comparison

## Machine Learning
- Demand forecasting

## Tableau Dashboard
- Inventory levels
- Stockout risk indicators
