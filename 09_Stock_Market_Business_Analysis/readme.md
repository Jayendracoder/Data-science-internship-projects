# Stock Market Business Analysis

## Project Overview
This project analyzes stock market data to understand price trends, volatility,
and investment patterns. It supports business and investment decision-making.

## Problem Statement
Investors and analysts require insights into historical price movements and
market behavior to manage risk and optimize returns.

## Dataset Description
- Stock symbol
- Open, High, Low, Close prices
- Volume
- Date

## Tools & Technologies
- Python (Pandas, NumPy)
- Time Series Analysis
- Machine Learning
- Tableau Dashboards

## Analysis Performed
- Price trend analysis
- Volume and volatility analysis
- Moving averages and returns

## Machine Learning
- Regression / Time-series forecasting
- Target: Stock price movement

## Tableau Dashboard
- Price trends
- Volume analysis
- Market performance indicators
