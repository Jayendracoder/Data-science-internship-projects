# Supermart Grocery Sales Analytics

## Project Overview
This project analyzes supermarket sales data to identify revenue drivers,
customer behavior, and product performance.

## Problem Statement
Retailers need sales insights to improve inventory management and profitability.

## Dataset Description
- Product category
- Sales
- Profit
- Discount
- Region

## Analysis Performed
- Sales & profit analysis
- Category-wise performance
- Discount impact analysis

## Machine Learning
- Sales prediction using regression

## Tableau Dashboard
- Sales by category
- Profit trends
- Regional performance
