# Daily Transactions Forecasting

## Project Overview
This project focuses on forecasting daily transaction volumes using historical
data to support planning and anomaly detection.

## Problem Statement
Accurate forecasting helps businesses manage cash flow and resources.

## Dataset Description
- Date
- Transaction count
- Amount

## Analysis Performed
- Time series analysis
- Trend & seasonality detection

## Machine Learning
- Time series forecasting models

## Tableau Dashboard
