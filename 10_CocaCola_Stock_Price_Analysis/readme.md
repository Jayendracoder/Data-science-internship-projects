# Coca-Cola Stock Price Analysis

## Project Overview
This project analyzes Coca-Cola stock price data to study long-term trends,
returns, and volatility. It provides insights for investors and financial analysts.

## Problem Statement
Understanding historical stock performance helps investors make informed
decisions and manage financial risk.

## Dataset Description
- Date
- Open, High, Low, Close
- Adjusted Close
- Volume

## Tools & Technologies
- Python (Pandas, NumPy)
- Financial Time Series Analysis
- Machine Learning
- Tableau Dashboards

## Analysis Performed
- Trend and return analysis
- Volatility measurement
- Moving average indicators

## Machine Learning
- Regression / Forecasting models
- Target: Closing price

## Tableau Dashboard
- Stock price trends
- Returns and volatility
- Long-term performance view
