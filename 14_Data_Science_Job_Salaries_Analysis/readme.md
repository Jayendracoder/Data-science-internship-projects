# Data Science Job Salaries Analysis

## Project Overview
This project analyzes data science job salaries based on role, experience,
location, and company size.

## Problem Statement
Job seekers and companies need market salary insights for fair compensation.

## Dataset Description
- Job title
- Salary
- Experience level
- Location
- Company size

## Analysis Performed
- Salary distribution
- Experience vs salary
- Role-wise comparison

## Machine Learning
- Salary prediction using regression

## Tableau Dashboard
- Salary by role
- Location-based salary comparison
