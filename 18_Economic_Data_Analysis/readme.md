# Economic Data Analysis

## Project Overview
This project analyzes economic indicators such as GDP, inflation, and
unemployment to understand macroeconomic trends.

## Problem Statement
Governments and analysts need economic insights for policy and forecasting.

## Dataset Description
- GDP
- Inflation rate
- Unemployment rate
- Year
- Country

## Analysis Performed
- Trend analysis
- Indicator correlation
- Economic growth patterns

## Machine Learning
- Economic forecasting models

## Tableau Dashboard
- GDP trends
