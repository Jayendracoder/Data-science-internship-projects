# Google Play Store Apps Analysis

## Project Overview
This project analyzes mobile app data from Google Play Store to understand
app ratings, downloads, categories, and pricing strategies.

## Problem Statement
App developers need insights to improve visibility, downloads, and ratings.

## Dataset Description
- App name
- Category
- Rating
- Reviews
- Installs
- Price

## Analysis Performed
- Rating vs installs analysis
- Category-wise app distribution
- Free vs paid apps comparison

## Machine Learning
- App success prediction

## Tableau Dashboard
- App category performance
- Rating distribution
